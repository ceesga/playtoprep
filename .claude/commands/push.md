Push de laatste commits van het playtoprep project naar GitHub.

1. Voer `git -C /Users/ceesgauw/claude_projecten/playtoprep status` uit om te controleren of er nog uncommitted wijzigingen zijn. Zijn die er, voer dan eerst /commit uit.
2. Voer `git -C /Users/ceesgauw/claude_projecten/playtoprep log origin/HEAD..HEAD --oneline` uit om te zien welke commits gepusht worden.
3. Vraag bevestiging aan de gebruiker voordat je pusht.
4. Push met `git -C /Users/ceesgauw/claude_projecten/playtoprep push`.
